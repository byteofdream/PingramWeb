Desktop Client Build

  • Qt6 Widgets + Network
  • CMake 3.20+
  • C++20 compiler
cd client
cmake -S . -B build
cmake --build build -j
./build/pingram_client

Backend Run

cd server
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

Keep backend running while testing the GUI client.
